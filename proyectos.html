Login + Registro con sesiones

Descripción

He desarrollado un sistema de registro y login implementando el patrón MVC y gestión de sesiones en PHP. El sistema almacena temporalmente la información de los usuarios mediante sesiones activas, incorporando validaciones de datos y un control básico de errores para garantizar la seguridad y funcionalidad del proceso de autenticación.

Tecnologías utilizadas

PHP - HTML - CSS

Qué aprendí

  • Implementar el patrón MVC en proyectos PHP
  • Gestionar sesiones de usuario
  • Validar datos de formularios y manejar errores
  • Organizar el código separando lógica y presentación

Prácticas de base de datos

Descripción

Durante el curso, he trabajado en el diseño y modelado de bases de datos relacionales, desarrollando bases de datos funcionales en MySQL a través de diversas prácticas. Las prácticas incluyen:

  • Creación de modelos entidad-relacón (MER)
  • Transformación a modelo relacional
  • Normalización de tablas para eliminar redundancias
  • Implementación inicial de bases de datos y tablas en MySQL

Tecnologías utilizadas

MySQL - Diagramas MER - Modelo relacional

Qué aprendí

  • Diseñar correctamente la estructura de una base de datos
  • Convertir un modelo conceptual a un modelo relacional
  • Aplicar principios de normalización para mejorar la eficiencia de la base de datos
  • Crear tablas y relaciones básicas en MySQL
