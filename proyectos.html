RESET ONG

Aplicación web creada para una ONG ficticia como proyecto de clase. Permite gestionar información sobre la organización, sus proyectos y sus usuarios, con diferentes roles según el tipo de acceso.

Desarrollada con PHP y conectada a una base de datos real, con una interfaz construida en HTML y Tailwind CSS, y pequeños toques de JavaScript para mejorar la experiencia de uso.

Un proyecto que me ha permitido trabajar por primera vez con una aplicación completa de principio a fin, combinando diseño, lógica de servidor y base de datos.

PHP HTML5 Tailwind CSS MySQL JavaScript
